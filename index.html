At the forefront
The simple content management system written in pure JavaScript
TaracotJS is a simple content management system (CMS) written in JavaScript on both client and server sides (using Node). It's free, open source and is running on multiple platrforms including Linux, MacOS and Windows. All modern browsers are supported.
The module structure: use one of existing modules or write your own
Currently available (and included): authorization/registration/profile module, file browser, file manager, menu control, pages editor, parts editor, fulltext search with morphology support, settings, text editor with syntax highlighting, user control, blog with comments support, social network authorization, social network core module.
Works on all modern devices: the responsive backend
No need for desktop computer to make changes on your website: all major mobile browsers (especially Google Chrome) are fully functional.
